**问题描述**

　　给定一个十进制整数*n*，输出*n*的各位数字之和。

<!--more-->

**输入格式**

　　输入一个整数*n*。

**输出格式**

　　输出一个整数，表示答案。

**样例输入**

20151220

**样例输出**

13

**样例说明**

　　20151220的各位数字之和为2+0+1+5+1+2+2+0=13。

**评测用例规模与约定**

　　所有评测用例满足：0 ≤ *n* ≤ 1000000000。

```c++
#include <cstdio>

int main() {
	int n, sum = 0;
	scanf("%d", &n);
	while (n > 0) {
		sum += n % 10;
		n = n / 10;
	}
	printf("%d\n", sum);
	return 0;
}
```
