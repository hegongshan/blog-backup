Java中的内部类可以分为以下四种：
* 成员内部类:作为外部类的成员定义
* 局部内部类:在外部类的方法中定义
* 匿名内部类（anonymous inner class）
* 静态内部类（static inner class）

内部类是一种编译器现象，与虚拟机无关。编译器将会把内部类翻译成文件名为“外部类名$内部类名.class”的class文件。
